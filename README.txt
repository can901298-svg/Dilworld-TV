Dilworld TV (Android TV Dashboard)

1) Android Studio (Hedgehog veya daha yeni) kur
2) Bu klasörü aç: DilworldTV
3) Build > Build APK(s)
4) APK çıktısı: app/build/outputs/apk/debug/app-debug.apk
5) APK'yı USB ile Android TV'ye kopyala ve kur (Bilinmeyen kaynaklar açık olmalı)

Notlar:
- Hava durumu: OpenWeather API key gömülü (Bayrampaşa).
- Döviz: TCMB today.xml (USD/EUR). Gram altın için TCMB XML içinde XAU varsa yaklaşık hesaplanır.
- Haber: AA RSS (cat=0) üzerinden başlıklar.
